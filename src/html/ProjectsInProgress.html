4. Little Trees

Little Trees is the code word for an online language teaching website. The website will be launched publically this year and will be targeted initially at teaching Chinese students to speak English.
Students and techers both need to login to the website and talk face to face with one another. We currently have three developers working on the project part-time including myself. As the website is quite complex we are using react, reactstrap and redux for the font-end. The backend is built on top of the Wordpress REST api. This combines all the skills I have learned up to now and pushes me further to learn the latest libraries and language features of javascript.

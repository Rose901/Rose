Truth lambs is the code word for a project still in progress. The website is ambitious because it has similar ideas to existing sites like Quora and Stack Exchange. It is being built by a team of volunteers in their spare time. I joined the team in 2018 and liked the vision of the website that allowed people to freely discuss spiritial questions and ideas. I offered my time to build the front-end for the website. My partner created the initial graphic designs and I implemented the website to be modern and responsive using mostly Bootstrap 4, Sass and Lightboxes.
